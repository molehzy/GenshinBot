#### 文件夹结构

```
│  GenshinBot.exe				- 编译好的程序
│  GenshinBot.exe.pid			- 保证同一路径下程序只能单开
│  menu.png						- 功能菜单图片
│
├─Data							- 存放全部数据的文件夹
│  │  Config.ini				- 程序配置文件
│  │  Group.ini					- 群开关配置文件
│  │
│  ├─Bind_db					- 存放绑定信息的文件夹
│  │
│  └─Img						- 缓存从API下载的图片的文件夹
│
└─Log							- 存放日志的文件夹
```
